Git og GitHub

Git er et verktøy som brukes for versjonskontroll og kodedeling. Det kan også kalles for en "Code Hosting Plattform" For eksempel så kan det vise oss historien til en kode, hvordan den har utviklet seg fra start til slutt, og da har muligheten til å gå tilbake til hvert punk i historien.
GitHub er en web-basert Git eller versons kontroll og en internett host service. Den er mest brukt for koding. Den tilbyr alt av det ditruberte versons kontrollen og SCM. GitHUb tilbyr altså kontroll over hvem som har tilgang til filene og flere sammarbeids funksjoner.
GitHub har en maskot som heter Octocat, som er en katt med fem tentakler og et menneske ansikt.
